# JadooTravels ✈️

A modern travel landing page built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.
Inspired by the Jadoo travel UI design.

## ✨ Features

* Responsive landing page design
* Smooth animations using Framer Motion
* Modern UI built with Tailwind CSS
* Optimized images using Next.js Image component
* Clean component-based architecture

## 🛠 Tech Stack

* **Framework:** Next.js
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Language:** TypeScript

## 🚀 Getting Started

Clone the repository:

```
git clone https://github.com/shbl007/JadooTravels.git
```

Navigate to the project folder:

```
cd JadooTravels
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
http://localhost:3000
```
